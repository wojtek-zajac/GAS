1. Get the company's LinkedIn logo (200x200px)

2. Log into your LinkedIn Developer account / My Apps / Create Application

3. Application Name = SR Company Name

  • Application Description = SR Company ID
  • Application Logo = 200px image
  • Application Use = Job and Recruiting
  • Website URL = http://www.smartrecruiters.com
  • Business Email = services@smartrecruiters.com
  • Business Phone = n-a
  • Acknowledge acceptance of terms of use and Submit

4. From the LinkedIn Developer page, click on JavaScript Settings

5. Add the following domains and click Update after entering each:

  • http://www.smartrecruiters.com
  • https://www.smartrecruiters.com

6. In the Authentication tab, ensure all checkboxes under Default Application Permissions are selected and click Update

7. Copy the Client ID (API Key) and Client Secret (API Secret) to the spreadsheet

8. Send the request to LinkedIn

9. Pour yourself a drink
